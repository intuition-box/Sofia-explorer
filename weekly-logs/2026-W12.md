# Weekly Log — Week 12 (March 13–20, 2026)

> Sofia Explorer went from zero to a fully functional behavioral reputation dashboard in one week.

## Project Creation
- Built the entire Sofia Explorer dashboard from scratch
- Defined the domain taxonomy, signal matrix, and platform catalog (142 platforms)
- Dark/light mode support

## Pages & Features
- **Feed** — Live activity feed with All Activity / My Circle toggle and intention filters
- **Profile** — Overview with top claims, interests grid, last activity, discovery scores, trust circle
- **Leaderboard** — Alpha testers ranking + season pool with PnL%
- **Vote** — Card-based voting on debate claims, support/oppose with staking
- **Streaks** — Streak leaderboard by consecutive daily certifications
- **Interest** — Domain deep-dive with niches, trending platforms, domain claims
- **Platforms** — Full catalog of 142 platforms with categories and connect buttons
- **Landing** — New landing page with Connect/Explore flow and feature cards

## Core Systems
- **Cart & Deposit** — Support/Oppose certification flow: choose intention → add to cart → select amount → deposit on-chain
- **ENS Resolution** — Multi-source ENS name + avatar resolution with fallbacks
- **Trust Circle** — Display user's trusted accounts with ENS avatars
- **Position Board** — Instant dialog with vault leaderboard per claim
- **Feed Processing** — Groups events by certifier, merges intentions, handles quest badges

## Profile & Scoring
- Profile header with ENS avatar, wallet address, Share on X
- Stats: domains, categories, platforms, signals count
- Discovery scores: Pioneer, Explorer, Contributor, Trusted
- Top claims filtered to only show Sofia-created claims
- Last activity from user's own certifications

## UX & Design
- Three-column layout: sidebar + main content + right sidebar
- Smooth transitions on all drawers and sidebars
- Page-enter animations
- Per-page decorative headers with color effects
- Animated loader during loading states
- Email/Google login as read-only mode, link wallet to interact

## Refactoring
- Clean architecture: services → hooks → components
- Separated styles into co-located CSS files
- Removed all dead code

## EthCC Integration
- EthCC 9 countdown in sidebar
- EthCC wallet linking to recover on-chain interest data
- Mapping of 100 EthCC topics + 17 tracks to Sofia domains

## Performance
- Optimized data loading from ~25s to ~3s on profile page
- Batch queries instead of sequential calls
- Fixed signals count to match the extension

## Key Decisions
- EthCC data = bonus points, not base scoring (not everyone has access to the event)
- Score = 0 if no real data (no fake scores)
- Top claims only from Sofia proxy (no unrelated claims)
